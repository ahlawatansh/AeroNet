About AeroNet

The app is a software demonstration of an embedded system designed to differentiate between drones and birds in real time. It captures live camera input, processes images using an AI/ML model, extracts object features, and classifies them as either a drone or a bird. When a drone is detected, the system displays a warning and sends an alert, demonstrating how an embedded surveillance system can improve security in sensitive areas.
